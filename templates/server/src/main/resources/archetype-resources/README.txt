How to create project from this archetype template.
2020-02-17
Tested on IntelliJ 2020.3

1- Create a new Maven project using option "create from archetype",
2- in the new appeared window enter following parameters:
  <groupId>com.devonfw.java.templates</groupId>
  <artifactId>devon4j-template-server</artifactId>
  <version>2020.12.002</version> <- latest release from GitHub repository
3- in next window enter your project name
4- wait until all dependencies will download and all directories will be created
5- the project should de ready to run
